About the project and the game

Esse projeto teve como proprósito educacional, ou seja, eu fiz apenas com o intuito de aprender a desenvolver um website por completo, incluindo banco de dados, backend e frontend.
O GuessCAT possui grande inspiração no site Termo, entretanto tem algumas variações do original.
- É necessário se cadastrar no site para poder jogar, como já informado, o site possui intuito educacional, isso inclui autenticação e autorização!
- Você pode jogar quantas vezes quiser, nao se prenda a jogar apenas um por dia!
