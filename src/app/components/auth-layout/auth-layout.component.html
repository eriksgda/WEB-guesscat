Welcome to GuessCAT Game!

How it Works:

  • Valid Characters: Disregard “Ç” and any accents (`, ^, ~, ´, etc.). Only words from the Portuguese language (pt-BR) are allowed.
  • Matching Length: Your guesses must have the same number of letters as the secret word.
  • Color-Coded Clues:
    • No Color: The letter is not in the secret word.
    • Yellow: The letter is in the word but in a different position.
    • Green: The letter is in the word and in the correct position.

How to Play:

  1. Sign Up or Log In: Create an account or log in to start playing.
  2. Start Game: Click the "Start Game" button in the sidebar to begin a new match.
  3. Enter Your Guesses: Input your word guesses, which must be the same length as the secret word.
  4. Analyze the Feedback: Use the color hints to refine your next guesses.
  5. Match History: Use the "Match History" button to view the games you’ve played.
