About the project and the game

Este projeto tem como proprósito o desenvolvimento técnico em desenvolvimento de websites, ou seja, foi feito apenas com o intuito de aprender a desenvolver um website por completo, incluindo banco de dados, backend e frontend, portanto, não possui nenhum fim lucrativo, somente educacional e para diversão.

O GuessCAT possui grande inspiração no site Termo,o qual você pode acesar a qualquer momento clicando AQUI, entretanto tem algumas variações do original, o que vale a pena ser explorado por você!

- É necessário se cadastrar no site para poder jogar, como já informado, o site possui intuito educacional, ou seja, isso inclui autenticação e autorização, já que é um dos temas mais importantes no desenvolvimento web!

- Você pode jogar quantas vezes quiser, nao se prenda a jogar apenas um por dia!
